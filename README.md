Assembly file: phone holder 2.FCStd

3D printed components:\
1.) Base 2 X1\
2.) Pivot X2 or Pivot3 X2 (Pivot3 has rounded edges for more comfort)\
3.) Holder 2 X1 or Holder 3 X1 (Holder 3 allow space for usb connection) 

Other components:\
1.) Bushing X2, example: https://www.aliexpress.com/item/1005005065448057.html \
2.) M6 nut X2\
3.) 140mm M6 stud X1\
4.) M6 "blind hole" knob X1, example: https://shopee.com.my/Rubber-Head-Nut-Hexagon-Torx-Nut-Star-Knob-Nut-Hand-Screw-Copper-Core-Plastic-Bolt-Cap-M3-to-M12-i.432361525.24020808638 \
5.) M6 "through hole" knob X1, example: https://shopee.com.my/Rubber-Head-Nut-Hexagon-Torx-Nut-Star-Knob-Nut-Hand-Screw-Copper-Core-Plastic-Bolt-Cap-M3-to-M12-i.432361525.24020808638 \
6.) 25mm M3 screw X6\
7.) M3 nut X6

Other information:\
1.) The M6 "blind hole" knob is to be screwed all the way into the stud such that they are fixed together. This knob is for controlling the angle of the phone. \
2.) It is recommended to stick rubber strips to the bottom of the base for added friction and stability.
